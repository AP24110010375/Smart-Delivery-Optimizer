# Smart-Delivery-Optimizer

## Project Description

Smart Delivery Optimizer is a real-world algorithm-based project developed in C.  
It simulates a delivery management system where orders are sorted, urgent deliveries are selected, packages are optimized, and the shortest delivery route is calculated.

This project demonstrates the practical use of important algorithms in real-life delivery applications such as food delivery, courier services, and e-commerce logistics.

## Algorithms Used

### 1. Sorting Algorithm
Orders are sorted based on priority so that high-priority orders are handled first.

### 2. Greedy Algorithm
The delivery system selects orders with the shortest delivery time first to improve efficiency.

### 3. Dynamic Programming
The 0/1 Knapsack algorithm is used to choose the most valuable packages that fit within the delivery bag capacity.

### 4. Dijkstra Algorithm
Dijkstra’s algorithm is used to find the shortest route from the warehouse to delivery locations.

## Real-World Application
